# Projeto de Carta Misteriosa (Mistery Letter)

![](color-guess.png)

Crie uma carta misteriosa a partir de um texto. 

Nesse projeto, foi utilizado:

* HTML
* CSS
* JavaScript

## Como funciona

Quando o usuário insere uma frase e clica no botão, ela é convertida em uma carta misteriosa, onde cada palavra tem uma estilização diferente. Caso queira mudar essa estilização, basta clicar na palavra.
